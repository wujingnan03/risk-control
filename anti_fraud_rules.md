## 规则详情

### FRD-001 多设备异常登录
- **条件**：近 7 天内使用 ≥ 3 台不同设备登录
- **严重程度**：高
- **风险说明**：正常用户通常使用 1-2 台设备。短时间内多设备登录可能是账号被盗或被他人操控
- **证据要求**：列出设备 ID、设备类型、首次出现时间
- **交叉验证**：结合 IP 是否同步变化。如果设备多但 IP 稳定，风险降低一级

### FRD-002 休眠账号突然激活
- **条件**：账号沉寂 ≥ 90 天后突然恢复活跃
- **严重程度**：中
- **风险说明**：长期休眠账号被激活可能是被盗号后用于欺诈活动
- **证据要求**：最后一次活跃日期、本次激活日期、间隔天数
- **交叉验证**：结合激活后是否立即发生交易行为。如果激活后仅浏览未交易，风险降低

### FRD-003 设备信息频繁变更
- **条件**：30 天内设备列表中 ≥ 50% 为新增设备
- **严重程度**：中
- **风险说明**：频繁更换设备可能是在隐藏真实设备指纹
- **证据要求**：新增设备列表、新增时间
- **交叉验证**：结合设备类型分布。如果均为同品牌手机，可能是正常换机

### FRD-004 凌晨异常活跃
- **条件**：0:00-6:00 时段登录占比 ≥ 30%
- **严重程度**：中
- **风险说明**：大量凌晨操作不符合正常基金投资者行为模式，可能是自动化脚本或异地（时差）操作
- **证据要求**：凌晨登录次数、占比、具体时间点分布
- **交叉验证**：结合 IP 是否为境外。若为境外 IP + 凌晨活跃，可能是正常时差；若为国内 IP + 凌晨活跃，风险提升

### FRD-005 登录频次突增
- **条件**：近 7 天日均登录次数 ≥ 历史日均的 3 倍，且日均 ≥ 10 次
- **严重程度**：中
- **风险说明**：登录频次陡增可能是自动化工具在探测或操控账号
- **证据要求**：历史日均、近期日均、增幅倍数
- **交叉验证**：结合是否伴随交易行为变化

### FRD-006 IP 地理位置频繁跳变
- **条件**：近 7 天登录 IP 涉及 ≥ 5 个不同省份
- **严重程度**：高
- **风险说明**：短时间内跨多省份登录在物理上不可能，强烈暗示使用代理/VPN 或账号被多人使用
- **证据要求**：IP 列表、对应省份、出现时间
- **交叉验证**：结合设备是否同时变化。IP 变但设备不变可能是 VPN；IP 和设备同时变则更危险

### FRD-007 境外 IP 登录
- **条件**：存在任意境外 IP 登录记录
- **严重程度**：高
- **风险说明**：国内基金销售平台用户从境外登录异常，可能涉及跨境欺诈或账号被盗
- **证据要求**：境外 IP 地址、归属国家/地区、登录时间
- **交叉验证**：结合用户注册信息中的证件类型。如果是港澳台居民或外籍人士，风险降低

### FRD-008 已知代理/VPN IP 段 [已禁用]
- **条件**：登录 IP 属于已知数据中心或 VPN 服务商 IP 段
- **严重程度**：中
- **状态**：已禁用
- **禁用原因**：当前无 IP 信誉库数据源，MCP 接口不提供 IP 类型（家庭/数据中心/VPN）判定能力，无法进行确定性匹配
- **风险说明**：使用代理/VPN 可能是为了隐藏真实位置
- **证据要求**：IP 地址、IP 类型判断依据
- **交叉验证**：结合其他网络异常指标
- **重新启用条件**：接入 IP 信誉库（如 IP2Location、MaxMind）后可重新启用

### FRD-009 注册后短时间内大额交易
- **条件**：注册后 7 天内发生单笔 ≥ 50 万或累计 ≥ 100 万交易
- **严重程度**：高
- **风险说明**：新注册用户快速大额交易是典型欺诈模式
- **证据要求**：注册日期、首笔交易日期、交易金额
- **交叉验证**：结合用户风险等级评定和适当性匹配

### FRD-010 交易模式突变
- **条件**：近期交易金额均值或频次与历史均值偏差 ≥ 200%
- **严重程度**：中
- **风险说明**：交易模式剧变可能暗示账号控制权发生变化
- **证据要求**：历史交易模式摘要、近期交易模式摘要、偏差程度
- **交叉验证**：结合是否同时出现设备或 IP 变化

### FRD-011 用户信息与设备/IP 地域不匹配
- **条件**：用户注册地址/证件归属地与主要登录 IP 所在省份不一致
- **严重程度**：中
- **风险说明**：可能是冒用他人身份
- **证据要求**：注册地信息、常用登录省份
- **交叉验证**：考虑工作地与户籍地不同的常见情况，仅当偏差严重时告警

### FRD-012 多维度同时触发
- **条件**：FRD-001 至 FRD-011 中**已启用**规则同时命中 ≥ 3 条
- **严重程度**：高
- **风险说明**：多个维度同时异常极大概率存在风险
- **证据要求**：列出所有命中规则及各自证据
- **交叉验证**：本规则本身即为交叉验证结果

## 规则维护说明

- 新规则通过工作流 B（特征提炼）生成建议，经人工审核后添加
- 规则 ID 格式：`FRD-{三位数字}`，从 013 起递增
- 每次更新需修改文件头部的版本号和更新日期
- 阈值参数可根据业务积累进行调整，调整需记录变更原因
