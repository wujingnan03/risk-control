# 雪球基金业务背景

## 平台概述

雪球基金销售平台是雪球旗下的基金销售业务，持有中国证监会颁发的基金销售牌照（由雪球基金销售有限公司运营）。平台主要面向个人投资者，提供公募基金的申购、赎回、转换、定投等交易服务。

## 业务类型

### 产品范围
- **公募基金**：股票型、混合型、债券型、货币型、指数型（含ETF联接）、QDII、FOF 等
- **私募产品**：如平台有私募代销资格，可能涉及阳光私募产品
- **基金组合**：雪球平台的基金组合跟投功能

### 核心交易类型
1. **申购（买入）**：用户购买基金份额
2. **赎回（卖出）**：用户赎回基金份额，资金回到银行卡
3. **基金转换**：同一基金公司旗下不同基金之间转换
4. **定投**：自动定期定额申购
5. **分红处理**：现金分红或红利再投资

### 资金流转
- 申购：银行卡 → 基金销售支付账户 → 基金公司
- 赎回：基金公司 → 基金销售支付账户 → 银行卡
- 资金到账一般 T+1 至 T+7 不等（视基金类型）

## 用户体系

### 用户注册与 KYC
- 实名注册：身份证 + 手机号 + 银行卡绑定
- 风险评测：根据监管要求完成投资者适当性评估
- 风险等级：保守型、稳健型、平衡型、积极型、激进型

### 用户画像特点
- 雪球社区用户为主，偏年轻化，有一定投资经验
- 社区讨论活跃度与基金交易行为可能相关
- 用户群体：散户为主，资金规模从几千到数百万不等

## 风控相关的业务特征

### 正常用户行为基线
- 设备：1-2 台（手机为主，偶尔 PC/平板）
- 登录频次：日均 1-5 次
- 登录时段：主要集中在 8:00-22:00
- 交易频次：月均 1-10 次
- IP：通常 1-3 个省份（居住地 + 工作地 + 偶尔出行）

### 需特别关注的业务场景
1. **货币基金快进快出**：货币基金 T+0/T+1 赎回特性，容易被用于资金过渡
2. **基金转换**：同一基金公司内的转换可能被用于混淆资金路径
3. **大额赎回后注销**：快速赎回全部资产后注销账号
4. **新注册即大额交易**：缺乏交易历史的新账号立即大额操作
5. **代理人操作**：同一人管理多个账号，可能涉及非法集资或洗钱

### 监管要求
- 《基金销售管理办法》
- 《证券期货投资者适当性管理办法》
- 《反洗钱法》及其配套规定
- 中国证监会、中国人民银行相关监管规则
- 基金销售机构需履行客户身份识别、可疑交易报告等反洗钱义务

## MCP 数据接口说明

### snowball-crm 可用接口

| 接口名 | 用途 | 关键返回字段 |
|--------|------|-------------|
| queryUserInfoByUid | 用户基本信息 | 姓名、证件号、手机号、注册时间、风险等级、银行卡列表 |
| queryLoginDeviceList | 设备列表 | 设备 ID、设备类型、品牌型号、首次/最近登录时间 |
| queryLoginLogList | 登录日志 | 登录时间、IP 地址、设备 ID、登录结果 |
| queryIpLocation | IP 归属地 | 省份、城市、运营商、IP 类型（家庭/数据中心） |

### 数据限制
- 登录日志：模式A默认查90天（100条），模式B/高风险升级至365天（150条），接口最大支持365天
- IP 归属地查询为实时接口，可能有频率限制
- 交易数据可能需要额外接口（当前 MCP 未提供，需通过用户信息接口间接获取）
- 跨账户关联查询当前不支持（AML-007、AML-009 相关规则可能无法完整验证）
